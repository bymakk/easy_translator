# Changelog

Здесь фиксируются заметные изменения расширения Easy Translator.

Формат вдохновлён [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [0.0.115] - 2026-04-25

### Изменено

- **Google Translate:** основной Google-перевод переведён на актуальный HTML-endpoint Google Translate; старый `client=gtx` оставлен резервным откатом.
- **Авторизация:** расширение теперь обращается к backend API через корректный домен `api.easytranslator.ru`, что устраняет ошибку входа из-за несовпадающего TLS-сертификата.
- **Popup-меню:** добавлено меню на панели браузера с включением/выключением переводчика и выбором режима запуска перевода.
- **README:** обновлено сравнение Google / Yandex / Grok на примерах, где AI-перевод лучше справляется со сленгом и контекстом.
- **Панель истории:** у первого элемента списка (`.translator-history__item`) отступ сверху **4px**; у последнего элемента отступ снизу **5px** — у списка появился небольшой «воздух» у верхнего и нижнего края.

Easy Translator

Перевод идёт напрямую из расширения на translate-pa.googleapis.com через неофициальный HTML-endpoint Google Translate. Если он временно недоступен, расширение откатывается на старый публичный endpoint translate.googleapis.com с client=gtx. Отдельный сервер и аккаунт Google не нужны.

Это не официальный платный Cloud Translation API: формат ответа и доступность могут измениться. Виджет «Google Website Translator» из solodev/google-translator подключает другой сценарий (скрипт element.js для всей страницы); здесь используется лёгкий запрос только для выделенного текста.

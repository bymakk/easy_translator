Translator Popup

Перевод идёт напрямую из расширения на translate.googleapis.com с параметром client=gtx (тот же класс неофициальных запросов, что используют простые клиенты перевода). Отдельный сервер и API-ключ не нужны.

Это не официальный платный Cloud Translation API: формат ответа и доступность могут измениться. Виджет «Google Website Translator» из solodev/google-translator подключает другой сценарий (скрипт element.js для всей страницы); здесь используется лёгкий запрос только для выделенного текста.
